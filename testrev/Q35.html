Question 35 C

Which element is used to group related form fields on a Webpage by creating a box around the related fields, to group and separate them?

A. legend.

B. datalist.

C. radio.

D. fieldset.

Choose your favorite Insect

Fieldset is used to group related elements in a form and the tag draws a box around the related elements.

A radio button is used to select one of the given options.

Legend is used to add a caption to a fieldset.

Datalist specify a list of pre-defined options and provide autocomplete feature for an input element.
